# Mosyrax: Simulador Interativo de Transistor MOS (NMOS-PMOS)
Mosyrax é uma aplicação educacional interativa composta por simuladores NMOS e PMOS em HTML, executados diretamente no navegador. Permite visualizar, em tempo real, o comportamento físico e elétrico de transistores MOS, integrando modelagem matemática, resposta elétrica e representação gráfica dinâmica.
